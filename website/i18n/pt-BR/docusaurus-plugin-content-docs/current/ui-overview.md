# Visão Geral da Interface do Player IPTV

## Estrutura do menu principal

O menu principal inclui várias seções:

- **Ao vivo**: assista a canais de TV ao vivo.
- **Recomendado para você**: canais sugeridos com base no histórico de visualização.
- **Minhas Listas**: gerencie suas playlists M3U adicionadas.
- **Opções**: configure as preferências do app.
- **Tendências**: canais populares ou em alta.
- **Filmes e Séries**: canais de filmes e séries, se disponíveis na sua região.
- **Buscar**: encontre canais por nome, conteúdo do EPG ou dentro de listas.

Cada categoria expande para mostrar submenus como **Guia de Programação**, **Mais opções** e **Editar Canal**.

## Tela de canais ao vivo

Ao assistir um canal, pressione **Seta para baixo** (teclado) ou faça dois swipes para cima para retornar ao menu principal.

Pressionar uma vez exibe os controles do player, como volume, velocidade de reprodução e troca de transmissão.

## Favoritos e histórico de exibição

- **Favoritos**: pressione o ícone de coração para adicionar um canal aos favoritos.
- **Favoritos com atalhos**: atribua números a canais favoritos para acesso rápido.
- **Histórico de exibição**: rastreia automaticamente os canais visualizados recentemente.
- **Continuar Assistindo**: disponível em **Ferramentas** > **Continuar Assistindo** para acessar o histórico do usuário.

Ambos são acessíveis na seção **Mais** ou diretamente na tela do player.

## Funcionalidade de busca

### Busca rápida
- **Pressione F3** a qualquer momento para abrir a busca
- **Clique em Buscar** na tela inicial
- **Tipos de busca**: Geral (padrão), conteúdo do EPG ou dentro de listas específicas

### Opções de busca
- **Busca geral**: pesquisa em todos os canais
- **Busca de EPG**: pesquisa no conteúdo do guia de programação
- **Busca em lista**: pesquisa dentro de listas IPTV específicas

## Navegando com controle remoto ou teclado

Use métodos de entrada padrão para navegar:

- Teclas de seta para mover entre menus.
- Enter/OK para selecionar.
- Voltar/Escape para voltar.
- Volume para cima/baixo para controlar o som.
- Duas vezes seta para baixo (ou swipe para cima duas vezes) para sair do modo tela cheia.

### Atalhos completos de teclado

| Ação | Atalho |
|--------|----------|
| Voltar | Backspace ou Esc |
| Navegar | Teclas de seta |
| Abrir canal favorito | Pressione o número do canal |
| Adicionar/remover dos favoritos | Ctrl + D ou Ctrl + Shift + D |
| Iniciar/parar gravação | F9 ou Ctrl + X |
| Histórico | Ctrl + H |
| Início | Home |
| Alternar idioma | Ctrl + L |
| Modo miniplayer | Alt + M ou Ctrl + M |
| Abrir URL | Ctrl + U ou Ctrl + O |
| Buscar | F3 ou Ctrl + F |
| Próximo vídeo | Ctrl + Seta para a direita |
| Vídeo anterior | Ctrl + Seta para a esquerda |
| Pausar/Play | Espaço |
| Recarregar transmissão | F5 |
| Avançar/Retroceder | Seta para a direita/esquerda (ocultar controles primeiro com seta para cima) |
| Sobre o Megacubo | F1 ou Ctrl + I |
| Tela cheia | F11 ou Alt + Enter |
| Ferramentas de desenvolvedor | F12 ou Ctrl + Shift + I |

### Personalizando atalhos

Vá em **Ferramentas** > **Key Mapping** para personalizar esses e outros atalhos.

## Controles do player

### Controles básicos
- **Play/Pause**: Espaço ou botão OK
- **Volume**: Seta para cima/baixo ou botões de volume
- **Tela cheia**: F11 ou clique duas vezes
- **Sair**: Escape ou botão Voltar

### Controles avançados
- **Trocar transmissão**: ícone de setas circulares
- **Faixa de áudio**: menu de três pontos > Mais opções > Selecionar áudio
- **Legenda**: menu de três pontos > Mais opções > Selecionar legenda
- **Qualidade do vídeo**: menu de três pontos > Mais opções > Selecionar qualidade
- **Configurações**: ícone de engrenagem

## Personalização de tema

### Temas integrados
1. Vá em **Ferramentas** > **Temas**
2. Selecione um tema disponível
3. Aplique as alterações instantaneamente

### Criando temas personalizados
1. Vá em **Ferramentas** > **Temas** > **Criar Tema**
2. Personalize:
   - Tamanho do texto e cores
   - Tamanho da grade e layout
   - Imagens de fundo
   - Cores dos elementos da interface
   - Aparência geral

### Recursos de tema
- **Personalização de texto**: ajuste tamanho e cores da fonte
- **Opções de layout**: modifique tamanho e espaçamento da grade
- **Imagens de fundo**: defina fundos personalizados
- **Esquemas de cores**: crie combinações personalizadas

## Dicas de navegação do menu

### Acesso rápido
- **Favoritos**: pressione o ícone de coração em qualquer canal
- **Recentes**: exibe automaticamente os canais assistidos por último
- **Busca**: pressione F3 ou use o ícone de busca
- **Atalhos numéricos**: pressione os números atribuídos aos canais favoritos

### Personalização
- **Temas**: personalize em Ferramentas > Temas
- **Atalhos de teclado**: mapeie teclas em Ferramentas > Key Mapping
- **Acessibilidade**: habilite leitor de tela em Opções > Comportamento

## Recursos de acessibilidade

- **Modo de alto contraste** para melhor visibilidade
- **Opções de texto grande** para leitura mais fácil
- **Navegação por teclado** para todas as funções
- **Suporte a leitor de tela** em dispositivos compatíveis
- **Modo narrador**: lê as opções e notificações selecionadas (Opções > Comportamento)

## Modos tela cheia e miniplayer

### Modo tela cheia
- **PC**: F11 ou Alt + Enter
- **Android**: modo padrão quando não está em miniplayer
- **Sair**: Escape ou botão Voltar

### Modo miniplayer
- **Ativação**: Alt + M ou Ctrl + M
- **Redimensionável**: ajuste o tamanho da janela
- **Sempre em cima**: opção para manter a janela visível
- **Disponível em**: PC e Android
