Perguntas Frequentes do Player IPTV

Perguntas Gerais​

O Megacubo é gratuito?​

Sim, o Megacubo é completamente grátis para usar. Também existe uma versão Premium que desbloqueia recursos adicionais como gravação, suporte a Chromecast e personalização de atalhos, mas o app permanece totalmente funcional sem ela.

O Megacubo é legal?​

Sim. O Megacubo não hospeda nem distribui conteúdo protegido por direitos autorais. Ele apenas reproduz listas IPTV adicionadas pelos usuários. Isso torna o aplicativo compatível com a maioria das leis relacionadas a software de streaming.

Ao usar o Modo Comunidade, o app compartilha apenas URLs de playlists M3U por meio de um sistema automatizado sem moderação humana. O Megacubo não hospeda, distribui ou verifica o conteúdo acessado por essas URLs. Os usuários são os únicos responsáveis por garantir que têm direitos adequados para acessar qualquer conteúdo e devem cumprir as leis e regulamentos aplicáveis em sua jurisdição.

Posso usar o Megacubo em dispositivos móveis?​

Sim! O Megacubo está disponível para:

  • Smartphones e TV Boxes Android
  • PCs com Windows, macOS e Linux

O Megacubo suporta legendas?​

Sim, mas apenas arquivos de legenda externos são suportados. Você pode selecioná-los clicando no menu de três pontos no canto inferior direito e escolhendo Mais opções > Selecionar legenda. Legendas incorporadas nas transmissões não são suportadas no momento. Não há opções de personalização da exibição de legendas.

Quais formatos de arquivo o Megacubo suporta?​

O Megacubo suporta:

  • Arquivos de playlist M3U/M3U8
  • Protocolos de streaming HLS e RTMP
  • Transmissões MPEG-DASH
  • Vários codecs de vídeo (H.264, H.265, etc.)

Suporte Técnico​

Por que alguns canais não funcionam?​

Alguns canais podem estar offline devido a problemas no servidor ou links expirados. Você pode tentar:

  • Clicar no ícone 🔄 de setas circulares nos controles do player para mudar de transmissão
  • Habilitar Modo Comunidade para buscar links alternativos compartilhados
  • Atualizar sua lista M3U

Meu app não inicia depois da atualização — o que devo fazer?​

Tente estes passos:

  1. Desative temporariamente qualquer antivírus ou firewall ativo
  2. Baixe e instale a versão mais recente do site oficial ou GitHub
  3. Se os problemas persistirem, desinstale completamente e reinstale o app
  4. Para usuários de PC, tente executar o instalador como administrador

Como atualizo o Megacubo?​

  • Atualizações manuais: baixe do site oficial
  • Verificar atualizações: visite o site oficial para as versões mais recentes
  • Instalação: instale sobre a versão existente para preservar as configurações

Nota: Para links de download e instruções de instalação, veja Instalação.

Recursos e Uso​

Como adiciono minha própria lista IPTV?​

  1. Vá em Minhas Listas > Adicionar Lista
  2. Insira a URL ou o caminho para seu arquivo M3U
  3. Clique em OK para importar

O app irá analisar a lista e exibir todos os canais disponíveis.

O que é o Modo Comunidade?​

O Modo Comunidade permite que usuários compartilhem anonimamente suas listas IPTV. Quando ativado, você pode acessar canais compartilhados por outros usuários sem precisar de suas próprias listas.

Como faço busca por canais?​

Pressione F3 a qualquer momento ou clique no ícone de busca na tela inicial. Você pode pesquisar por:

  • Geral: em todos os canais
  • EPG: no conteúdo do guia de programação
  • Listas: dentro de listas IPTV específicas

Como configuro favoritos com atalhos?​

  1. Adicione um canal aos favoritos
  2. Atribua um atalho numérico a ele
  3. Pressione a tecla numérica para acessá-lo rapidamente

Como habilito suporte a Chromecast?​

O suporte a Chromecast é um recurso Premium. Após atualizar:

  1. Clique no ícone de transmissão no player
  2. Selecione seu dispositivo Chromecast
  3. Transmita diretamente para a TV

Posso gravar TV ao vivo?​

A funcionalidade de gravação está disponível na versão Premium:

  • Iniciar/Parar: F9 ou Ctrl + X
  • Armazenamento local: gravações são salvas no seu dispositivo
  • Gerenciamento de arquivos: organize gravações localmente

Como personalizo a interface?​

Vá em Ferramentas > Temas para:

  • Mudar temas e cores
  • Ajustar tamanho do texto e layout
  • Definir imagens de fundo
  • Criar temas personalizados

Nota: Para informações detalhadas sobre personalização da interface, veja User Interface.

Como mudo a qualidade de vídeo?​

Clique no menu de três pontos no canto inferior direito e selecione Mais opções > Selecionar qualidade. Se essa opção estiver ausente, a transmissão possui apenas um nível de qualidade.

Aspectos Legais​

Quais são as responsabilidades legais dos usuários?​

Os usuários são responsáveis por garantir que suas listas IPTV sejam obtidas legalmente. O Megacubo não recomenda nem endossa streaming ilegal.

Posso compartilhar minhas listas IPTV com outras pessoas?​

Sim, o Megacubo permite compartilhar listas por meio do Modo Comunidade. No entanto, sempre certifique-se de que as listas compartilhadas não contenham material protegido por direitos autorais ou violem leis locais.

Por que o Megacubo solicita permissão para acessar a internet?​

O aplicativo precisa de acesso à internet para:

  • Baixar e atualizar listas IPTV
  • Buscar dados de EPG (programação)
  • Acessar listas compartilhadas da comunidade
  • Verificar atualizações e melhorias

Meu histórico de exibição é privado?​

Sim, seu histórico de exibição é armazenado localmente no seu dispositivo e não é compartilhado conosco ou com outros usuários, a menos que você habilite explicitamente o Modo Comunidade.

Desempenho e Otimização​

Como posso melhorar o desempenho de streaming?​

  • Use conexão cabeada em vez de Wi-Fi
  • Habilite o Modo para Dispositivos Lentos em aparelhos antigos
  • Feche outros aplicativos para liberar recursos
  • Verifique sua velocidade de internet (mínimo 200KBps a 2MBps por transmissão)

Nota: Para dicas de otimização, veja Performance.

Por que a qualidade do vídeo está ruim?​

A qualidade do vídeo depende de:

  • Sua conexão com a internet
  • A qualidade da transmissão origem
  • O poder de processamento do seu dispositivo
  • A congestão de rede

Como reduzir buffering?​

  • Mude para outra transmissão usando o ícone de setas circulares
  • Habilite modo de desempenho nas configurações
  • Limpe o cache e reinicie o app

Como reduzir o uso de memória?​

  • Habilite o modo para dispositivos lentos
  • Carregue menos listas ao mesmo tempo
  • Limpe o cache regularmente
  • Feche outros aplicativos

Comunidade e Suporte​

Como posso reportar um bug?​

  1. Verifique issues existentes no GitHub
  2. Crie uma nova issue com informações detalhadas
  3. Inclua informações do sistema e logs de erro
  4. Descreva os passos para reproduzir o problema

Como posso contribuir com o Megacubo?​

Existem várias maneiras de contribuir:

  • Reportar bugs e sugerir recursos
  • Traduzir o aplicativo para o seu idioma
  • Ajudar outros usuários na comunidade
  • Contribuir com código no GitHub

Onde posso obter ajuda?​

  • Consulte o FAQ e a documentação
  • Envie e-mail para contact@megacubo.tv
  • Abra uma issue no GitHub
  • Participe do GitHub Discussions
  • Visite nosso site: megacubo.tv

Nota: Para informações completas de suporte e solução de problemas, veja Support & Contact e Troubleshooting.

Estamos aqui para ajudar! A comunidade Megacubo é amigável e acolhedora. Não hesite em nos contatar se precisar de assistência.
