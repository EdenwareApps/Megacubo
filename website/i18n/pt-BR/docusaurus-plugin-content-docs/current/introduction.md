# Megacubo IPTV Player - Introdução

## O que é o Megacubo?

**Megacubo** é um **player IPTV gratuito e de código aberto** que permite assistir canais de TV ao vivo de todo o mundo usando arquivos de lista de reprodução **M3U**. Suporta várias plataformas, incluindo **player IPTV para Windows**, **app IPTV para macOS**, **software IPTV para Linux**, **player IPTV para Android** e **streaming para TV Box**.

Ao contrário de serviços de streaming tradicionais, o Megacubo não hospeda nem fornece conteúdo diretamente. Em vez disso, ele atua como um player para listas IPTV externas fornecidas por usuários ou fontes de terceiros.

## Principais recursos do player IPTV

- **Player IPTV gratuito**: aplicativo de streaming de TV ao vivo totalmente gratuito, sem assinaturas necessárias
- **IPTV de código aberto**: código-fonte completo disponível no GitHub para a comunidade de desenvolvimento IPTV
- **Aplicativo IPTV multiplataforma**: player nativo IPTV para Windows, macOS, Linux, Android e TV Boxes
- **Modo de comunidade IPTV**: acesse milhares de listas IPTV compartilhadas e canais ao vivo da comunidade
- **Suporte a listas de reprodução M3U**: importe e gerencie listas de reprodução M3U com EPG (Guia Eletrônico de Programação)
- **Otimização de streaming IPTV**: configurações avançadas para dispositivos com hardware limitado, buffer e otimização de rede
- **Interface de player IPTV**: navegação intuitiva com suporte a controle remoto e atalhos de teclado
- **Modo mini player IPTV**: imagem na imagem enquanto multitarefa
- **Favoritos e histórico IPTV**: salve canais favoritos e histórico de visualização
- **Controles parentais IPTV**: filtro de conteúdo para uso familiar
- **Pesquisa e categorias IPTV**: encontre canais IPTV rapidamente com pesquisa e categorização
- **Suporte a transmissão IPTV**: transmita ao vivo para Chromecast e outros dispositivos compatíveis
- **Retomar reprodução IPTV**: continue assistindo de onde parou
- **Conformidade legal IPTV**: streaming IPTV seguro - reproduz apenas conteúdo de fontes fornecidas pelo usuário
- **Atalhos para favoritos**: atribua números de atalho aos favoritos para acesso rápido
- **Personalização de temas**: personalize a aparência com temas integrados ou crie o seu próprio
- **Modo portátil**: execute o aplicativo a partir de drives USB ou armazenamento externo

## Requisitos técnicos

### Requisitos de rede
- **Largura de banda mínima**: 200KBps a 2MBps por stream
- **Recomendado**: conexão de internet estável para desempenho ideal

### Requisitos do sistema

| Plataforma | Status | Requisitos mínimos |
|---------|--------|---------------------|
| Windows | ✅ Totalmente suportado | Windows 7, 4GB de RAM |
| macOS   | ✅ Totalmente suportado | macOS 10.13, 4GB de RAM |
| Linux   | ✅ Totalmente suportado | Ubuntu 16.04+, 4GB de RAM |
| Android | ✅ Totalmente suportado | Android 7.0, 2GB de RAM |
| TV Boxes | ✅ via instaladores APK | Android 7.0+ |

### Permissões do Android IPTV
O player IPTV para Android requer as seguintes permissões para streaming ideal:
- **Acesso à internet** para streaming de TV ao vivo e canais IPTV
- **Acesso ao armazenamento** para cache de conteúdo e arquivos M3U
- **Wake lock** para reprodução contínua sem desligar a tela
- **Acesso ao estado da rede** para monitorar a conexão de streaming

### Formatos IPTV compatíveis
- **Listas M3U/M3U8** - formato padrão de listas IPTV
- **Formato estendido EXTM3U** - formato aprimorado de IPTV com metadados
- **Streams HLS** - HTTP Live Streaming para IPTV
- **Múltiplos codecs de vídeo** - H.264, H.265, VP8, VP9 para streaming IPTV
- **Suporte a EPG** - Guia Eletrônico de Programação

Você pode baixar a versão mais recente do player IPTV Megacubo no site oficial ou no GitHub. Veja [guia de instalação](installation.md) para instruções detalhadas de configuração.
