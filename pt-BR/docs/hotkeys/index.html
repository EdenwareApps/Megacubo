Atalhos padrão no Megacubo:​


Abaixo está uma lista de atalhos de teclado disponíveis no Megacubo. Para saber mais sobre as opções de configuração do aplicativo, clique aqui.

  • Voltar: Backspace ou Esc

  • Navegar: teclas de seta

  • Abrir canal favorito: pressione o número do canal (veja os números no setor de "Favoritos")

  • Adicionar/remover canal dos favoritos: Ctrl + D ou Ctrl + Shift + D

  • Iniciar/parar gravação: F9 ou Ctrl + X

  • Histórico: Ctrl + H

  • Início: Home

  • Alternar idioma: Ctrl + L

  • Modo miniplayer: Alt + M ou Ctrl + M

  • Abrir URL: Ctrl + U ou Ctrl + O

  • Buscar: F3 ou Ctrl + F

  • Próximo vídeo: Ctrl + Seta para a direita

  • Vídeo anterior: Ctrl + Seta para a esquerda

  • Pausar/Play: Espaço

  • Recarregar transmissão: F5

  • Avançar/Retroceder: Seta para a direita/esquerda (oculte os controles do player pressionando a seta para cima primeiro)

  • Sobre o Megacubo: F1 ou Ctrl + I

  • Alternar tela cheia: F11, / ou Alt + Enter

  • Console de Ferramentas de Desenvolvimento: F12 ou Ctrl + Shift + I


Se houver um atalho que você gostaria de ver adicionado, conte para nós.

Nota: Para habilitar recursos especiais no app, você pode ir em Ferramentas > "Key Mapping" para personalizar esses e outros atalhos.
