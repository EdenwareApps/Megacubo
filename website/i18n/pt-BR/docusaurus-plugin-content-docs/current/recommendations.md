# Sistema de Recomendações

## Como as recomendações de conteúdo funcionam

O Megacubo analisa:

- Canais que você assistiu
- Canais que você favoritou
- Tempo gasto em cada canal
- Padrões de visualização e preferências

Em seguida, sugere conteúdo similar na seção **Recomendado para você**.

## Recurso Continuar Assistindo

Acesse seu histórico em **Ferramentas** > **Continuar Assistindo**:

- **Canais recentes**: mostra o conteúdo assistido recentemente
- **Recomendações melhores**: o histórico é usado para aprimorar as sugestões
- **Acesso rápido**: retome a exibição de onde você parou
- **Padrões de visualização**: ajudam o algoritmo a aprender suas preferências

## Personalizando suas preferências

1. Vá em **Opções** > **Comportamento** para ajustar as configurações de recomendação.
2. Ajuste categorias como esportes, notícias, filmes etc.
3. Suas recomendações serão atualizadas de acordo.

## Limpando histórico de visualização

Para redefinir as recomendações:

1. Vá em **Opções** > **Avançado** > **Opções do desenvolvedor**.
2. Limpe o cache e redefina as configurações.
3. As recomendações serão recalculadas na próxima vez que você assistir um canal.

## Entendendo o algoritmo

O sistema de recomendações usa vários fatores:

### Padrões de visualização
- **Canais assistidos recentemente**
- **Categorias mais vistas**
- **Duração de exibição**
- **Preferências de horário**

### Interações do usuário
- **Canais favoritados**
- **Conteúdo avaliado**
- **Canais ignorados**
- **Comportamento do Continuar Assistindo**

### Similaridade de conteúdo
- **Combinação de gênero**
- **Preferências de idioma**
- **Relevância geográfica**
- **Categorias de canal**

### Preferências manuais
- **Favoritos** para melhorar as recomendações
- **Avalie o conteúdo** para ajudar o algoritmo
- **Ignore conteúdo indesejado** para evitar sugestões semelhantes
- **Use Continuar Assistindo** para aprimorar o aprendizado

## Privacidade e dados

- **Processamento local**: recomendações são calculadas no seu dispositivo
- **Sem compartilhamento de dados**: seu histórico de visualização permanece privado
- **Opção de desativar**: você pode desligar as recomendações completamente
- **Retenção de dados**: o histórico é armazenado localmente e pode ser apagado a qualquer momento
- **Sem rastreamento online**: nenhuma informação pessoal é enviada para servidores

## Integração com o Modo Comunidade

> **Nota**: Para informações detalhadas sobre o Modo Comunidade, veja [Community Mode](community-mode.md).

### Seleção de listas
As listas da comunidade são escolhidas com base em:
- **Relevância de conteúdo** para suas preferências de exibição
- **Algoritmo de recomendação** compatível
- **Qualidade e confiabilidade da lista**
- **Feedback e avaliações de usuários**

## Recursos avançados

### Filtragem inteligente
- **Filtragem de conteúdo** com base em controles parentais
- **Filtragem de qualidade** para transmissões preferidas
- **Filtragem de idioma** para conteúdo no idioma desejado
- **Filtragem regional** para conteúdo local

### Melhorias de aprendizado
- **Aprendizado contínuo** a partir dos seus hábitos de visualização
- **Sugestões adaptativas** que melhoram com o tempo
- **Ajustes sazonais** para conteúdo baseado em época do ano
- **Integração de tendências** para conteúdo popular

## Solução de problemas das recomendações

### Problemas comuns
- **Sem recomendações**: verifique se há histórico de visualização
- **Sugestões ruins**: limpe o histórico e comece do zero
- **Conteúdo inadequado**: ajuste os controles parentais
- **Problemas de idioma**: verifique as configurações de idioma

### Soluções
1. **Assista mais conteúdo** para criar histórico
2. **Favoritar canais preferidos** para melhorar as sugestões
3. **Limpar e redefinir** os dados de aprendizado, se necessário
4. **Ajustar preferências** nas configurações de recomendação

---

*O sistema de recomendações melhora com o tempo à medida que você usa o app. Quanto mais você interagir, melhor serão as sugestões personalizadas.*
